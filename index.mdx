## Sistema Premium de Gestión de Talleres de Bicicletas

### ¿Qué es?

Servana Academy es una plataforma integral diseñada para talleres de bicicletas premium que buscan digitalizar y profesionalizar sus operaciones. Combina gestión de servicios, inventario, equipo y clientes en una experiencia minimalista y eficiente.

## Nuestra Misión

Transformar la manera en que los talleres de bicicletas premium gestionan su negocio, eliminando procesos manuales, mejorando la transparencia con los clientes, y aumentando la eficiencia operativa.

## ¿Para quién?

Diseñado para talleres premium que buscan ofrecer una experiencia de servicio excepcional, con control total sobre sus operaciones, diagnósticos profesionales y comunicación fluida con sus clientes.

## Características Principales

### 🚴 Gestión Integral de Servicios
Control completo del ciclo de vida de cada servicio, desde la reserva hasta la entrega final y feedback del cliente.

### 🔍 Diagnósticos Inteligentes
Sistema de aprobaciones con fotos, presupuestos automáticos y comunicación directa con el cliente antes de realizar trabajos adicionales.

### 💳 Pagos Integrados
Integración completa con Shopify para gestión de productos, servicios, pagos en línea y facturación automática.

### 📱 Notificaciones Automáticas
Comunicación multicanal (WhatsApp, Email) para mantener al cliente informado en cada etapa del servicio.

### 📲 Mobile-First
Diseño optimizado para dispositivos móviles, permitiendo gestionar el taller desde cualquier lugar.

### 📊 Analytics y Reportes
Panel de control en tiempo real con KPIs, métricas de desempeño y reportes detallados para tomar mejores decisiones.

## Nuestro Enfoque

- **🎯 Simplicidad Premium**: Diseño minimalista inspirado en las mejores marcas tech
- **🔧 Gestión Integral**: Todo lo que necesitas en un solo lugar
- **📱 Mobile-First**: Optimizado para tablets y smartphones
- **🔐 Seguridad**: Permisos granulares y autenticación robusta
- **🔗 Integraciones**: Shopify, WhatsApp, Email y más

## Tecnología

- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Auth, Storage)
- **E-commerce**: Shopify Admin API
- **Notificaciones**: WhatsApp Business API + Email
- **Hosting**: Lovable Cloud
- **Estado**: React Query (TanStack Query)
